# Concept Drift Datasets v1.0
 These are the concept drift datasets we made.
## Background
Concept drift is a phenomenon in which the statistical properties of a target domain change over time in an arbitrary way[1]. Concept drift problem exists in many real-world situations, such as sensor drift and the change of operating mode.
